## 1. Tonalität-Regeln — Do's und Don'ts

### Die Grundregel

Schreib so, wie ein Freund spricht, der sein Lieblings-Gym empfiehlt. Direkt, konkret, ehrlich — ohne Schleimerei und ohne Marketing-Aufgeblasenheit.

### Du-Ansprache (immer, ausnahmslos)

| Falsch (Sie) | Richtig (Du) |
|---|---|
| „Wir freuen uns, Sie bei uns begrüßen zu dürfen." | „Komm einfach vorbei." |
| „Unsere Mitglieder schätzen unser vielseitiges Angebot." | „Unsere Mitglieder kommen wegen der Sauna — und bleiben wegen der Gemeinschaft." |
| „Vereinbaren Sie jetzt Ihr Probetraining." | „Buch dein Probetraining." |

### Satzlänge

- Maximum 15–20 Wörter pro Satz
- Lieber zwei kurze Sätze als ein langer
- Gedankenpause durch Gedankenstriche — statt Schachtelsätze

**Falsch:**
„Unser breit gefächertes Angebot an inkludierten Leistungen, welches neben einer modernen Geräteausstattung auch einen exklusiven Saunazugang, zahlreiche Live-Kurse sowie eine persönliche Ernährungsberatung umfasst, bietet Ihnen eine einzigartige Rundum-Betreuung."

**Richtig:**
„Sauna, 10+ Live-Kurse, Getränkeflat, Trainingsplan — alles drin. Kein Aufpreis."

## 2. Verbotene Floskeln — mit besserem Ersatz

Jede dieser Formulierungen wird auf der Fitness Factory Website **nicht** verwendet. Die Spalte rechts zeigt den konkreten Ersatz.

| Verbotene Floskel | Warum problematisch | Besserer Ersatz |
|---|---|---|
| „Erlebe dein neues Ich" | Abgedroschen, verspricht vage Transformation ohne Substanz | „Was sich in 8 Wochen verändert — du weißt es erst danach." |
| „Dein Partner für Fitness" | Jedes Studio behauptet das | „Wir kennen dich beim Namen." |
| „Entdecke die Möglichkeiten" | Nichtssagend, kein konkreter Nutzen | Konkret nennen, was möglich ist: „Test die Sauna. Mach einen Spinning-Kurs. Frag nach dem Trainingsplan." |
| „hochwertig" | Adjektiv ohne Beweis | Zeigen statt behaupten: „Geräte der Marke [TBD]", „Sauna direkt im Studio" |
| „innovativ" | Bedeutet im Fitness-Kontext gar nichts | Weglassen. Was neu ist, konkret benennen. |
| „ganzheitlich" | Wellness-Bingo | „Sauna, Ernährungsberatung, Trainingsplan — alles an einem Ort." |
| „State-of-the-art" | Klingt wie 2005 | Weglassen. Oder konkrete Geräte nennen. |
| „einzigartig" | Behaupten tut das jeder | Den echten Unterschied konkret nennen: „Keine andere Kette in Hattingen hat kostenlose Sauna inklusive." |
| „wir bieten Ihnen" | Sie-Ansprache + passiv | „Du bekommst", „dabei ist", „inklusive" |
| „maßgeschneidert" | Abgenutzte Worthülse | „Wir erstellen dir einen Plan, der zu dir passt" |
| „rundum" | Unpräzise Auffüllung | Stattdessen konkret aufzählen |
| „Fitness-Community" | Klingt nach Franchise-Broschüre | „Die Leute, die hier trainieren, kennen sich." |
| „motivierendes Ambiente" | Adjektiv-Stapel | Zeigen: Gallerie-Bilder, konkrete Saunadaten |
| „Wohlfühlatmosphäre" | Beliebig | Konkret: „familiär, kein anonymes Studio" |
| „nachhaltig" | Buzzword ohne Kontext | Nur verwenden wenn es stimmt und konkret belegt wird |

## 3. CTA-Formeln — Was für Fitness Factory funktioniert

### Grundprinzip: Niedriger Einstieg, klares Ziel

Kein Mensch unterschreibt blind einen Jahresvertrag. Der erste CTA ist immer der niedrigschwelligste Schritt: Probetraining, Anruf, WhatsApp.

### CTA-Hierarchie

```
1. Probetraining buchen (niedrigste Hürde)
2. Per WhatsApp schreiben
3. Anrufen
4. Preise ansehen / Mitglied werden (höchste Hürde)
```

### Bewährte CTA-Formeln für Fitness Factory

**Primär-CTAs (Probetraining):**
- `Probetraining buchen`
- `Komm zum Probetraining`
- `Einen Tag kostenlos trainieren`
- `Per WhatsApp Probetraining buchen`

**Sekundär-CTAs (Kontakt):**
- `Jetzt anrufen: 02324 33777`
- `Auf WhatsApp schreiben`
- `Frag uns einfach`

**Tertiär-CTAs (Info):**
- `Preise ansehen`
- `Alle Kurse im Überblick`
- `Mehr über das Team`

### Gebot: Immer Kontext vor dem CTA

Niemals ein Button einfach hinwerfen. Der Kontext macht den CTA relevant:

**Falsch:**
`Jetzt Mitglied werden [Button]`

**Richtig:**
„Noch unsicher? Erst kostenlos testen — dann entscheiden. `[Probetraining buchen]`"

### WhatsApp-CTA — immer mit vorausgefüllter Nachricht

Der WhatsApp-Link für Probetraining:
```
https://wa.me/4915737580001?text=Hallo%2C%20ich%20m%C3%B6chte%20ein%20Probetraining%20buchen.
```

Der WhatsApp-Link für allgemeine Anfragen:
```
https://wa.me/4915737580001?text=Hallo%2C%20ich%20habe%20eine%20Frage%20zur%20Mitgliedschaft.
```

Der WhatsApp-Link für die Fremdgeh-Aktion:
```
https://wa.me/4915737580001?text=Hallo%2C%20ich%20interessiere%20mich%20f%C3%BCr%20die%20Fremdgeh-Aktion.
```

### Dringlichkeits-Formeln (sparsam einsetzen)

Nur echte Knappheit kommunizieren — keine falschen Fristen.

**Erlaubt (wenn wahr):**
- „Aktion gültig, solange das Angebot besteht."
- „Fremdgeh-Aktion: Jetzt nutzen, solange du noch woanders Mitglied bist."

**Verboten:**
- „Nur noch 3 Plätze!"
- „Angebot endet morgen!"
- „Jetzt oder nie!"

## 4. SEO-Keyword-Integration — Natürlich und wirksam

### Kern-Keywords (Priorität 1)

| Keyword | Monatliches Suchvolumen (geschätzt) | Einsatzorte |
|---|---|---|
| Fitnessstudio Hattingen | Hoch (Haupt-KW) | H1, Meta-Title, erste 100 Wörter, Alt-Tags |
| Gym Hattingen | Mittel | H2 oder Fließtext, Meta-Description |
| Fitness Hattingen | Mittel | Fließtext, FAQ-Antworten |
| Probetraining Hattingen | Mittel | Probetraining-Seite H1, Meta-Title |
| Studio Hattingen | Niedrig | Fließtext, Footer |

### Long-Tail-Keywords (Priorität 2)

| Keyword | Einsatzorte |
|---|---|
| Fitnessstudio Hattingen-Holthausen | Kontakt-Section, Google Business |
| Fitnessstudio Hattingen günstig | Preis-Section, FAQ |
| Sauna Fitnessstudio Hattingen | Leistungen-Section, FAQ |
| Kurse Fitnessstudio Hattingen | Kursplan-Seite |
| Yoga Hattingen | Kursplan, Meta Kursplan-Seite |
| Spinning Hattingen | Kursplan, Meta Kursplan-Seite |
| Zumba Hattingen | Kursplan |
| Probetraining Fitnessstudio Hattingen | Probetraining-Seite |

### Einzugsgebiet-Keywords (Priorität 3 — für lokales SEO)

Natürlich einbinden in Kontakt-Section, Footer oder dedizierte Einzugsgebiet-Absätze:
- Fitnessstudio Bochum-Linden
- Fitnessstudio Sprockhövel
- Fitnessstudio Witten
- Fitnessstudio Blankenstein
- Gym Hattingen-Holthausen

### Keyword-Integration: Konkrete Regeln

**Regel 1: Keywords müssen natürlich klingen**

Falsch: „Als Fitnessstudio Hattingen bieten wir Fitnessstudio Hattingen Leistungen."
Richtig: „Das Fitnessstudio in Hattingen, das Sauna und Kurse inklusive hat."

**Regel 2: Kern-Keyword in diesen Positionen**
- Title-Tag: immer am Anfang, z.B. `Fitnessstudio Hattingen | Fitness Factory ...`
- H1: mindestens 1x Hattingen (muss aber natürlich klingen)
- Erste 100 Wörter der Seite: 1-2x natürlich eingebaut
- Meta-Description: 1x, mit Value Proposition
- Alt-Tags: `Gerätebereich Fitnessstudio Hattingen` etc.

**Regel 3: Semantische Cluster aufbauen**

Nicht nur das Exact-Match-Keyword, sondern verwandte Begriffe:
- Gym, Studio, Sportstudio, Trainingsraum (für "Fitnessstudio")
- Sauna, Wellness, Entspannung (für "Sauna inklusive")
- Yoga, Spinning, Kurs, Live-Kurs (für Kursangebote)
- Holthausen, Hattingen, NRW, Ruhrgebiet (für lokales SEO)

**Regel 4: FAQ ist SEO-Gold**

Fragen in der FAQ-Seite = exakte Suchanfragen von echten Nutzern. Jede Frage ist ein potenzielles Featured Snippet.

Gute FAQ-Fragen für Featured Snippets:
- „Was kostet ein Fitnessstudio in Hattingen?"
- „Welches Fitnessstudio in Hattingen hat Sauna?"
- „Gibt es ein kostenloses Probetraining in Hattingen?"

## 5. Headline-Formeln — Fitness Factory spezifisch

### Formel 1: Kontrast-Headline (Gegensatz schafft Spannung)

Schema: `[Erwartung brechen] — [Wahrheit liefern]`

Beispiele:
- `Dein Studio. Keine Ausreden.` (Hero)
- `Menschen, keine Maschinen.` (Team)
- `Echte Betreuung. Kein Vertriebler.`
- `Alle inklusive. Kein Aufpreis.`

### Formel 2: Fakten-Headline (Zahlen überzeugen)

Schema: `[Zahl] + [konkreter Nutzen]`

Beispiele:
- `Ab 35€. Alles drin.`
- `10+ Kurse. Null Aufpreis.`
- `3 Monate. 0€. Kein Haken.` (Fremdgeh-Aktion)
- `Mo–Fr bis 23:00 Uhr.` (Öffnungszeiten als Headline)

### Formel 3: Direkte Ansprache-Headline

Schema: `Dein/Deine + [konkretes Ding]`

Beispiele:
- `Dein Trainingsplan.`
- `Dein Kursplan.`
- `Dein Preis. Deine Laufzeit.`
- `Deine erste Woche.`

### Formel 4: Problem-Lösung (implizit)

Schema: `[Problem benennen] → [Lösung eingebaut]`

Beispiele:
- `Keine Wartezeit. Keine versteckten Kosten.`
- `Sauna ohne Aufpreis — kein Extraprogramm nötig.`
- `Parkplätze direkt vor der Tür — kein Stadtparkhaus.`
- `Kein Verkaufsgespräch. Nur Training.`

### Formel 5: Community-Headline (inklusiv, einladend)

Schema: `Alle [Gruppe] willkommen` oder implizite Inklusion

Beispiele:
- `Von 18 bis 70+. Alle trainieren hier.`
- `Anfänger. Fortgeschrittene. Alle Level.`
- `Du musst nichts beweisen.`
- `Einfach mitkommen.`

### Formel 6: Action-Headline (kurz, imperative Energie)

Schema: `Verb + [minimale Info]`

Beispiele:
- `Komm einfach vorbei.`
- `Trainier einen Tag mit uns.`
- `Wechsel jetzt.` (Fremdgeh-Aktion)
- `Schau dich um.` (Galerie)
