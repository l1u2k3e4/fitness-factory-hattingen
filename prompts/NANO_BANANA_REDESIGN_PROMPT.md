# Nano Banana Pro — Redesign-Prompt für Fitness Factory Hattingen

> **Funktion:** Redesign (Screenshots der aktuellen Website hochladen)
> **Ziel:** Inspiration generieren, wie die neue Website aussehen könnte
> **Technik:** Image Transformation (Technique 8) + Structured (Technique 2)

---

## Anleitung

1. Öffne **Nano Banana Pro** → Wähle **Redesign**
2. Lade die **Screenshots der aktuellen Website** hoch (Homepage, Probetraining, Preise etc.)
3. Kopiere den passenden Prompt unten in das Textfeld
4. Generiere und iteriere

---

## PROMPT 1: Homepage-Redesign (Gesamtansicht)

```
Redesign this fitness studio website into a clean, professional, modern layout.

Design direction:
- Background: Pure white (#FFFFFF) with alternating light gray (#F7F7F8) sections
- Primary color: Bold red (#C8102E) — used ONLY for CTA buttons and key accents
- Headlines: Black (#1A1A1A), bold sans-serif font (Inter or Manrope style)
- Body text: Dark gray (#374151)
- Cards: White background with thin light gray border (1px #E5E7EB), subtle shadow

Layout rules:
- Clean white background throughout — NO dark/gray theme
- Sharp corners: buttons have 4px radius, cards have 6px radius — NO rounded-full, NO pill shapes
- Generous whitespace between sections — sections breathe
- NO glow effects, NO gradient text, NO gradient buttons
- NO colored shadows, NO pulsing animations
- Buttons are flat red with white text, clean hover state (darker red)
- Navigation: white background with thin bottom border, logo left, links center, red CTA button right

Sections from top to bottom:
1. Top bar: black background, white text — opening hours, phone, WhatsApp
2. Navigation: white, clean, minimal — logo left, links center, red "Probetraining" button right
3. Hero: Full-width image with dark overlay, white headline "Dein Fitnessstudio in Hattingen — alles inklusive", red CTA button
4. Trust bar: light gray background, Google rating, member count, 3 feature icons
5. Services: 8 feature cards in a grid — white cards, thin border, red icon, black title, gray description
6. Pricing: 3 pricing cards side by side — standard card highlighted with red border and "Beliebteste Wahl" badge
7. CTA section: solid red background, white text "Komm vorbei. Kostenlos. Unverbindlich.", white button
8. Team: trainer photo cards on white background — photo, name, specialty
9. Course schedule: compact weekly table on light gray
10. Gallery: 6 photos in clean grid
11. Testimonials: quote cards with stars
12. FAQ: clean accordion with thin dividers
13. Contact: split layout — info left, form right
14. Footer: black background, white text, 4 columns

Style reference: Professional European web agency design. Think Apple.com clarity meets fitness energy. NOT a template, NOT AI-generated looking. A human designer made this.

Aspect ratio: 9:16 vertical (full page scroll view)
No watermarks, no placeholder text like "Lorem ipsum".
```

## Tipps für die besten Ergebnisse

1. **Lade die Screenshots der aktuellen Website hoch** — Nano Banana nutzt sie als Basis für das Redesign
2. **Starte mit Prompt 1** (Gesamtansicht) für den ersten Eindruck
3. **Iteriere:** Wenn etwas zu dunkel/rund/generisch aussieht, ergänze im Prompt: "Make it cleaner, sharper corners, more whitespace, less AI-looking"
4. **Variationen:** Ändere einzelne Farben oder Layouts und generiere erneut
5. **Kombiniere:** Nutze Prompt 5 (Collage) um mehrere Ansichten auf einmal zu sehen
6. **Negativ-Prompts hinzufügen** falls nötig:
   ```
   No dark background, no gradient buttons, no glow effects, no rounded-full shapes, no AI-typical design patterns, no Bootstrap aesthetic
   ```
